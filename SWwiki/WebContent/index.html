About

What is this?

The Star Wars Wiki began as a practice project about consuming a REST API (SWAPI) and now is Jabba's favourite online wiki about the Star Wars universe!

How can I support this?

With small donations we can keep the wiki running for free, please consider throwing us some beer money to say thank you. With every $10 we can keep the site up for another month! This project is open source and you can contribute on GitHub.
